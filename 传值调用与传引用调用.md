## 传引用调用：
如果形参是引用类型，它将绑定在对应的实参上，修改形参的时候同时修改实参。通常在实参为字符串，拷贝需要大量内存是能起到节约内存和加速的作用。
如下：
```
void reset(int &a){
  i = 0;
}

int j =42;
reset(j);
cout<<"j = "<<j<<endl;
```

## 传值调用：
非传引用调用就是传值调用。将实参的值拷贝后赋给形参。

## 指针形参：
指针形参也是传值形参，它将实参的指针拷贝一份，拷贝后实参与形参是两个不同的指针，但是却指向同样的对象，所以可以间接访问所指对象和修改它的值。
如下:
```
void reset(int *p){
  *p = 0;
}

int i = 42;
reset(&i);
cout<<"i = "<<i<<endl; //输出i=0；
```
